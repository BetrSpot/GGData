GGData
============

GGData is a simple persistent data storage class for the Corona SDK.

Example Usage
-------------------------

Coming soon.

Features
-------------------------

* Unlimited storage boxes to allow for easy separation of data.
* Uses both Json and SQLite to obfuscate the data slightly.
* Easy score tracking.
* Simple API.
